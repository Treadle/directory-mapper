# directory-mapper
This little script helps visualize a given directory structure.

## Sample output
```
Enter absolute path: C:\test
└── a
   ├── file_a.txt
   ├── file_b.txt
   ├── folder_b
   │  ├── file_c.txt
   │  └── folder_f
   │     └── file_e.txt
   ├── folder_c
   ├── folder_d
   └── folder_e
      └── file_d.txt
```
